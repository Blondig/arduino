# Arduino

Utility library files for Arduino

## Description

Arduinoまわりの便利クラスをまとめています。

## Features

***DN7C3JA020 : ***
PM2.5センサをそこはかとなく使えるようにした

***PPD42NS : ***
ほこりセンサをpulseInではなく非同期で使えるようにした

***GP20U7 : ***
GPSモジュールをSoftwareSerialで使えるようにした

## Requirement

- Arduino IDE 1.6.5
